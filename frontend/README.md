# 前端项目结构设计

- `src/views`：页面级视图（登录、申报列表、申报表单、审核、统计、用户管理）。
- `src/components`：可复用组件（状态标签等）。
- `src/api`：Axios 接口封装，按模块划分。
- `src/router`：路由与权限守卫。
- `src/store`：Pinia 状态管理（鉴权与当前用户）。

## 页面说明

1. 登录页：工号+密码登录。
2. 申报列表页：查询、分页、跳转新建/编辑。
3. 申报表单页：标准申报字段、保存、提交、驳回重提。
4. 审核页：待审核列表 + 审核动作弹窗。
5. 统计页：按任务查询、Excel导出、归档导出。
6. 用户管理页：增删改查、重置密码、多选权限。
